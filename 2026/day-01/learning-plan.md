## Day 01: Linux Basic Commands
- **Bash** stands for *Bourne Again Shell*.
- It’s a command-line interpreter (or shell) widely used in Unix-like operating systems, especially Linux.
- Bash allows us to:
  - Create, delete, edit, view, and modify files & directories
  - Change properties of files & directories
  - Perform many more operations via the command line
